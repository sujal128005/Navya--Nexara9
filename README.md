# Project Navya 🚗
**Status:** In Development (Qualification Round - IRoC-U 2026)

## Overview
Navya is an end-to-end autonomous driving system designed for the ISRO Robotics Challenge (URSC 2026). The project integrates multi-sensor fusion to navigate complex environments autonomously.

## Core Features
* **Perception:** Real-time object detection and classification.
* **Localization:** Precision mapping using NavIC-integrated systems.
* **Control:** PID-based steering and velocity control for smooth trajectory following.

## Tech Stack
* **Software:** ROS2, C++, Python, OpenCV.
* **Hardware:** Jetson Nano/Orin, LiDAR, Depth Cameras.
